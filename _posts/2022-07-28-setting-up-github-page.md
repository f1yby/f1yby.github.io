Using Minimal Mistake Template, Github Page is close at your hand.

## Using [Minial Mistake](https://github.com/mmistakes/minimal-mistakes) Template

1. Go to [minimal mistake starter](https://github.com/mmistakes/mm-github-pages-starter), and then click `Use this template`.
2. Enter `Repository name`. To use Github Page, type `${GITHUB_USERNAME}.github.io`. Such as `LWSG.github.io`
3. Edit `_config.yml` to describe your site and yourself.
4. [Optional] Add `google_site_verification: "${GOOGLE_VERIFICATION_CODE}"` to `_config.yml` to tell google your control on your site. Plus, the **sitemap** is on `${DOMAIN}/sitemap.xml`

**Done!**

## General Setup

For general setup, you can refer to [Github Page Official Website](https://pages.github.com/) to set up your own pages.
